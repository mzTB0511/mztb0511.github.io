    Lambda 使用

    遍历List集合，修改元素值

    List<String> list = Arrays.asList("张三","李四","王五");
List newList = list.stream().map(item -> "你好" + item).collect(Collectors.toList());
System.out.println(JSONObject.toJSONString(newList));
Json 输出 ["你好张三","你好李四","你好王五"]

    遍历List集合，重构元素结构，生成新的 List

    List<String> list = Arrays.asList("张三","李四","王五");
List<Map> newList = list.stream().map(item -> {
    Map<String,String> itemMap = new HashMap(0);
    itemMap.put("id","1");
    itemMap.put("name",item);
    return itemMap;}
    ).collect(Collectors.toList());

System.out.println(JSONObject.toJSONString(newList));
Json 输出[{"name":"张三","id":"1"},{"name":"李四","id":"1"},{"name":"王五","id":"1"}]
